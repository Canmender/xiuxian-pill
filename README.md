# XiuXianPill v1.0

MC修仙服丹药插件 — 经验丹服用系统。

## 功能

- **26种经验丹**：体修 13 种 + 法修 13 种
- **境界服用限制**：只能服用 ≤ 当前境界的丹药
- **右键服用**：手持丹药右键即可使用
- **管理员命令**：/pill give、/pill list、/pill use
